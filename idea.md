# Project Name: LogicKart

## Problem Statement
Students struggle to find a single platform that provides logical learning toys, puzzles, and beginner-friendly coding resources.

## Solution
LogicKart is an e-commerce website that sells logical toys, puzzles, and learning kits to help students improve problem-solving skills.

## Target Users
- School Students  
- College Students  
- Parents  

## Features
- Product Listing  
- Search & Filter  
- Cart System  
- Checkout  
- Login/Signup  
- Order History  

## Tech Stack
- React  
- Tailwind CSS  

## Future Scope
- Payment Gateway  
- Recommendation System  
- Admin Dashboard  
- Reviews & Ratings
